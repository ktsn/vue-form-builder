Vue From Builder

Overview

Vue Form Builder helps you to make a large number of forms regarding your model.

Simplest Example

{{ user.name }}

<form-for v-model="user" name="user">
  <text-field for="name"></text-field>
  <p>{{ user.name }}</p>
</form-for>
new Vue({
  data: {
    user: {
      name: ''
    }
  }
})
